Description

GRATIN-SM est une solution compléte pour les sociétés de service. C'est une application FULL WEB qui offre des fonctionnalités basées sur les conventions faites avec vos partenaires et prestataires.

- Grâce à GRATIN-SM, vous pourrez :

      * Diminuer le temps de traitements.

      * Gérer vos clients.

      * Gérer vos prestataires de service.

      * Gérer vos carnets d’adresses avec toutes les informations sur les clients, prestataires de service et les conventions.

      * Gérer les Devis reçu.

      * Calculer automatique les commissions.

      * Accèder à l’historique de vos clients.

      * Automatiser la création des factures.

      * Proposition de choix de prestataire selon les critère prés définis.

      * Gérer vos dépenses.

      * Générer des rapports et tableaux de bord.

      * Suivre les paiements.

      * Suivre les écheances des clients (via un calendrier bien détaillé).

      * Suivre les impayés client.

      * Envoyer des factures rapidement (par email ou pour imprimer).

      * Gestion des accées.

      * Générer des statistiques.

Avec GRATIN-SM vos services sont plus fiables, préçues et surtout à bas prix.
