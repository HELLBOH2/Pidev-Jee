Description

GRATIN-SMS est une solution compléte d'envoi de SMS pour vos clients.

La diffusion de vos remises, vos offres, votre activité, etc sera plus facile via notre nouvelle application WEB d'envoi de SMS.

L'application peut être intégrer directement à votre base de donnée afin d'informer vos client. Nous possédons aussi une base de donnée contenant une large catégorie de personne regrouper selon leur activité professionnel, région, age, sexe...).

Notre application GRATIN-SMS est une solution FULL WEB donc elle peut être utiliser via n'importe quel support (PC, Smartphone, Tablette, ...) et à n'importe quel endroit (Maison, Bureau, ...)

NB : Le frais d'un SMS est 0.050 dt (Même qu'un SMS envoyer par mobile). On utilise une carte SIM de l'un des opérateurs Tunisien, donc vous peuvez acheter un forfait pour réduire le coût d'un SMS.
